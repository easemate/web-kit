# @easemate/web-kit Demo

Interactive demo application for `@easemate/web-kit` components.

## Running the Demo

```bash
# From the workspace root
npm run dev

# Or from this directory
cd apps/demo
npm run dev
```

The demo will be available at [http://localhost:3003](http://localhost:3003).

## Available Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start development server |
| `npm run build` | Build for production |
| `npm run preview` | Preview production build |
| `npm run typecheck` | Run TypeScript type checking |

## About

This demo showcases the components available in `@easemate/web-kit`:

- **Controls**: Sliders, toggles, inputs, dropdowns, color pickers
- **Layout**: State panels, fields, popovers
- **Advanced**: Curve editor, code highlighting, FPS monitor

For full documentation, see the [core package README](../../packages/core/README.md).
